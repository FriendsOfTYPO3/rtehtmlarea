============
Contribution
============

Feel free to submit any pull request, or add documentation, tests, as you please.
We will publish a new version every once in a while, depending on the amount of
changes and pull requests submitted.

If you want to keep adding features, and keep HtmlArea compatible with the latest
browser versions, and TYPO3 versions, feel free to contact `Markus Klein`_.

.. _Markus Klein: markus.klein@typo3.org

License

Authors
=======

A lot of contributors have been working on this area while this functionality
was part of the TYPO3 Core. This package is now maintained by a loose group of
TYPO3 enthusiasts inside the TYPO3 Community.
